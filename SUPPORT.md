# AreaFlow Support

## Supported Versions

Before `v1.0.0` is published, the `main` branch is the only supported development line. After publication, the latest `v1` minor receives fixes; the previous minor receives critical security fixes for 90 days.

## Support Channels

- Security issues follow `SECURITY.md`.
- Reproducible product defects use the GitHub bug template.
- Operational incidents use the incident template and must omit credentials, tokens, project content and personal data.

Support does not include operation of external OIDC, PostgreSQL, S3, TLS, load balancers or observability providers.

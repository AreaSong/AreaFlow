# v0.5 Runner Preview

阶段合同见 [`../architecture/v0.5-runner-preview-contract.md`](../architecture/v0.5-runner-preview-contract.md)。
本阶段只建立 dry-run execution model 和 DB-only run control；不打开真实 worker execution、
engine 调用、项目写入或 `./task-loop run` forwarding。

## Goal

建立 execution/run 模型，但只做 dry-run。

## Includes

- Task schema。
- Copy/verify prompt model。
- Execution queue model。
- Attempt model。
- Engine adapter interface。
- Permission preflight。
- Runner preview CLI/API。
- Dry-run events and audit events。

## Excludes

- 真实调用 AI engine。
- 修改项目文件。

## Success

- `run`、`run_task`、`run_attempt` 能表达 execution queue、copy、verify、repair 和 checkpoint。
- Permission preflight 能拒绝未授权路径、命令、secret 和 network。
- Dry-run 能生成 events、audit_events 和 generated artifacts。
- verify fail 不会 mark done。
- risk gate 能阻断 high / mission-critical task。
- Runner preview 不写被管理项目文件。

## v0.5a Boundary

v0.5a 先落地 dry-run runner preview：

- Migration `000005_v0_5_runner_preview.sql` 增加 `run_tasks`、`run_attempts`，并扩展 `runs`。
- `areaflow run preview <project> <version>` 创建 dry-run run/task/attempt 记录。
- REST API 提供 `POST /api/v1/projects/{project_key}/workflow-versions/{version}/runner-preview`。
- Preview 只创建 `copy` 和 `verify` dry-run attempt，不调用 AI engine，不执行命令，不写被管理项目。
- Preview 生成 `runner_preview_report` artifact 原文到 local artifact store，并在 PG `artifacts` 中保存 metadata、URI、hash、size 和 content type。
- CLI JSON 与 API response 都返回 `artifacts[]`，供 Web/Desktop 后续展示 dry-run evidence。
- Permission preflight 记录 write、command、secret、network 均未尝试。
- high / mission-critical risk 在 `risk_policy != allow` 时 blocked。
- `run.start`、`run.drain` 和 `run.cancel` 只控制 dry-run run 的 AreaFlow DB 状态，并在 command response
  中记录不领取 task、不启动 worker、不执行命令、不调用 engine、不使用 secret、不访问 network。
- Command response 必须记录 `area_matrix_write_attempted=false`，并且不能把 preview 结果解释为
  AreaMatrix execution cutover。

## v0.5a Verification

```bash

git diff --check -- README.md docs internal migrations
```

PostgreSQL smoke 应证明：

```text
run.status = passed
tasks[0].status = queued
run.dry_run = true
len(tasks) = 1
len(attempts) = 2
len(artifacts) = 1
artifacts[0].artifact_type = runner_preview_report
artifacts[0].sha256 非空
high risk + pause policy 被 runner preview blocked
run.start/drain/cancel command_requests completed
run.start/drain/cancel task_claimed = false
run.start/drain/cancel worker_started = false
non-dry-run run control 被 blocked
area_matrix_write_attempted = false
```

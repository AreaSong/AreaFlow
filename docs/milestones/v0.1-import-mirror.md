# v0.1 Import + Status Mirror

## Goal

AreaFlow 能注册 AreaMatrix，导入 workflow/residual/task metadata，写入 PostgreSQL，并生成粗略状态。

## Contract

本阶段最小 CLI、API、读写边界、status projection shape 和 v0.2 handoff 以
[`../architecture/v0.1-import-mirror-contract.md`](../architecture/v0.1-import-mirror-contract.md) 为准。
AreaMatrix adapter 的只读导入深度、minimum import set 和 artifact metadata 策略以
[`../architecture/areamatrix-import-scope-contract.md`](../architecture/areamatrix-import-scope-contract.md) 为准。

## Includes

- Project registration。
- AreaMatrix adapter v0.1。
- Import run。
- Artifact metadata。
- Residual import。
- `.areaflow/status.json` export。

## Current Progress

- `areaflow migrate up` 已可应用 v0.1 core schema。
- `areaflow project add` 已可注册 AreaMatrix 示例配置。
- `areaflow project import` 已可只读导入 AreaMatrix workflow versions、residuals、artifact metadata 和 status snapshot。
- `areaflow project export-status` 已可按 `write_status` 权限显式导出 `.areaflow/status.json`。
- `areaflow project status-projection-apply` 是长期主语义，`export-status` 是 compatibility command。

## Excludes

- Task execution。
- Workflow cutover。
- Web/Desktop。
- AI engine invocation。

## Success

AreaFlow 能识别 AreaMatrix v1-mvp complete、formal alpha blocked、active tasks 0、backlog 5 closed / 0 open。
